How to run the project

The Microbit Code

There are two methods:


Method 1:

1. Copy the javascript code to the online editor at this link.

2. Install the required extensions ("datalogger" and "microbit-pxt-timeanddate")

3. Either run in the simulator or download the hex file, connect a microbit to the computer and drag the hex file to the microbit on your file explorer (or connect through webusb).

Javascript code: "Artefact/microbitCodeJavascript.js"


Method 2:

1. Get the hex file I provided

2. Drag it onto the simulator at this link. or onto a physical microbit connected to your computer through the file explorer.

Hex file: "Artefact/microbitCodeHex.hex"


The Python Code

Similarly, there are two methods.


Method 1:

1. Install python and the required libaries ("csv" and "statistics")

2. Run the program with the python interpreter program. (Please ensure the csv file and exectuable are in the same directory!)

Python file: "Artefact/dataAnalysisPython.py", Data file: "Artefact/sleepdata.csv"


Method 2:

1. Run the executable provided. (Please ensure the csv file and exectuable are in the same directory!)

Executable file: "Artefact/executableDataAnalysisPython.exe", Data file: "Artefact/sleepdata.csv"
